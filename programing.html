Line Following

Line following is a method of autonomous movement where a robot follows a line drawn on the map.

Line Squaring

Line squaring is a method where both driving motors move accordingly to the color sensors value to align the robot to a spectacular line on the map.

Line Detection

Line detection is a method that the robot detects a line on the map using the color sensor of the robot.
